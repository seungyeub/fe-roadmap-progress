## 1. 상태(Status) 규칙

- `TODO`  : 아직 시작하지 않음
- `DOING` : 현재 진행 중
- `DONE`  : 1회 학습/정리 완료
- `REVIEW`: 학습은 했지만, 복습이 필요하거나 더 깊게 들어가야 할 상태

## 4. 업데이트 방식

- **새로운 섹션을 공부하기 시작**할 때
  - 표에 **해당 섹션의 상태를 `DOING`**으로 변경
  - 가능하면 `대표 날짜`와 `학습 로그 링크`도 바로 기록
- **섹션 1회 학습 완료** 시
  - 상태를 `DONE`으로 변경
  - 대표 날짜와 로그 링크를 채워 넣기
- **복습이 필요**하다고 느끼면
  - 상태를 `REVIEW`로 변경
  - `study_schedule.md`에 해당 섹션 복습 주차를 다시 잡기

이 문서는 “지금 내가 어디까지 와 있는지”를 확인하는 **메인 대시보드** 역할.
가능하면 **하루에 한 번 또는 최소 주 1회**는 열어서 상태를 업데이트.
